## 📌 Assignment 1: Responsive Dashboard

- Created a responsive dashboard layout
- Sidebar navigation using Bootstrap
- Displayed statistics using cards
- Mobile-friendly design

### 🛠 Technologies Used:
- HTML
- CSS
- Bootstrap

## 📌 Assignment 2: User Registration System

- User registration form
- Data stored using Local Storage
- AJAX POST request using Fetch API
- Display registered users on a separate page

### 🛠 Technologies Used:
- HTML
- CSS
- JavaScript
- AJAX (Fetch API)
